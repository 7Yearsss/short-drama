# Home Banner Carousel Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let admins mark published series as home-page "banner" items with a manual sort order, and show them on the public homepage as an iOS-app-switcher-style horizontal carousel of portrait posters, replacing the current static VIP promo card.

**Architecture:** Two boolean/int fields on `Series` (`isHomeBanner`, `bannerOrder`) drive everything — no new tables. The existing generic admin `PATCH /api/admin/series/:id` already forwards arbitrary body fields straight into `prisma.series.update`, so it needs no route-logic change, only a type update. A new public `GET /api/series/banners` endpoint serves the public carousel. The admin series workbench gets a small settings panel; the public homepage gets a new `HomeBanner` client component using native `scroll-snap` + a `scroll` listener to compute per-card scale/opacity (no new npm dependency).

**Tech Stack:** Fastify + Prisma + Postgres (existing), Next.js 14 client component, plain CSS (`globals.css`), Vitest for API tests, Node `node:test` static-regex tests for the web app (matches this repo's existing web test style — see `apps/web/test/*.test.mjs`).

## Global Constraints

- 后端改动先写测试再实现 (write the backend test before the implementation) — see [CLAUDE.md](../../../CLAUDE.md).
- 小步提交,每个 commit 只做一件事 — one focused commit per task below.
- 不引入范围外的抽象/功能 — no new banner-ordering table, no carousel npm dependency, no unlimited banner cap enforcement (explicitly out of scope per the approved spec).
- Do not modify `apps/web/tsconfig.json` by hand; if `next build`/`next dev` auto-edits it, run `git checkout -- apps/web/tsconfig.json` before committing (see [CLAUDE.md](../../../CLAUDE.md)).
- Only `status: 'published'` series are eligible to display as banners; no cap on how many can be marked (per approved spec at [docs/superpowers/specs/2026-07-10-home-banner-design.md](../specs/2026-07-10-home-banner-design.md)).
- The existing VIP membership promo card (`.featured-row` / `.membership-card` in `apps/web/app/page.tsx`) is removed entirely, not relocated.

- [ ] **Step 4: Create the `HomeBanner` component**

Create `apps/web/components/HomeBanner.tsx`:

```tsx
'use client';

import { useEffect, useRef, useState } from 'react';
import Link from 'next/link';
import { fetchHomeBanners, BannerSeries } from '@/lib/api-client';

const AUTOPLAY_INTERVAL_MS = 4500;
const RESUME_DELAY_MS = 3000;

export default function HomeBanner() {
  const [banners, setBanners] = useState<BannerSeries[] | null>(null);
  const [scales, setScales] = useState<number[]>([]);
  const trackRef = useRef<HTMLDivElement>(null);
  const autoplayTimer = useRef<ReturnType<typeof setInterval>>();
  const resumeTimer = useRef<ReturnType<typeof setTimeout>>();
  const activeIndex = useRef(0);

  useEffect(() => {
    fetchHomeBanners()
      .then(setBanners)
      .catch(() => setBanners([]));
  }, []);

  function recomputeScales() {
    const track = trackRef.current;
    if (!track) return;
    const cards = Array.from(track.children) as HTMLElement[];
    const center = track.scrollLeft + track.clientWidth / 2;
    setScales(
      cards.map((card) => {
        const cardCenter = card.offsetLeft + card.offsetWidth / 2;
        const ratio = Math.max(0, 1 - Math.abs(center - cardCenter) / track.clientWidth);
        return 0.85 + ratio * 0.15;
      })
    );
  }

  function stopAutoplay() {
    if (autoplayTimer.current) clearInterval(autoplayTimer.current);
    autoplayTimer.current = undefined;
  }

  function startAutoplay() {
    stopAutoplay();
    if (!banners || banners.length <= 1) return;
    autoplayTimer.current = setInterval(() => {
      const track = trackRef.current;
      if (!track) return;
      const cards = Array.from(track.children) as HTMLElement[];
      activeIndex.current = (activeIndex.current + 1) % cards.length;
      const target = cards[activeIndex.current];
      track.scrollTo({
        left: target.offsetLeft - (track.clientWidth - target.offsetWidth) / 2,
        behavior: 'smooth',
      });
    }, AUTOPLAY_INTERVAL_MS);
  }

  function scheduleResume() {
    if (resumeTimer.current) clearTimeout(resumeTimer.current);
    resumeTimer.current = setTimeout(startAutoplay, RESUME_DELAY_MS);
  }

  useEffect(() => {
    const track = trackRef.current;
    if (!track || !banners || banners.length === 0) return;
    recomputeScales();
    track.addEventListener('scroll', recomputeScales, { passive: true });
    return () => track.removeEventListener('scroll', recomputeScales);
    // eslint-disable-next-line react-hooks/exhaustive-deps
  }, [banners]);

  useEffect(() => {
    startAutoplay();
    return stopAutoplay;
    // eslint-disable-next-line react-hooks/exhaustive-deps
  }, [banners]);

  if (!banners || banners.length === 0) return null;

  return (
    <section className="home-banner" onPointerDown={stopAutoplay} onPointerUp={scheduleResume}>
      <div className="home-banner-track" ref={trackRef}>
        {banners.map((banner, index) => (
          <Link
            key={banner.id}
            href={`/series/${banner.id}`}
            className="home-banner-card"
            style={{
              transform: `scale(${scales[index] ?? 0.85})`,
              opacity: scales[index] ? 0.4 + ((scales[index] - 0.85) / 0.15) * 0.6 : 0.6,
              backgroundImage: banner.coverUrl ? `url(${banner.coverUrl})` : undefined,
            }}
          >
            <span className="home-banner-title">{banner.title}</span>
          </Link>
        ))}
      </div>
    </section>
  );
}
```
